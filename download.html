What to Expect

Once released, the Chatyour client will be available as a Python-based desktop application for Windows. It requires Python 3 and the wxPython library to run.

The client connects to the Chatyour server via WebSocket and provides a fully accessible chat experience through:

  • A wxPython graphical interface designed for screen reader compatibility
  • Built-in text-to-speech via accessible_output2 for instant audio feedback
  • Keyboard shortcuts for all major actions (Ctrl+Q to leave a room, Ctrl+S to save)
  • Automatic background reconnection on network interruption
  • Support for multiple simultaneous chat rooms

System Requirements

  • Windows operating system (7 or later)
  • Python 3.8 or higher
  • wxPython 4.x
  • A screen reader such as NVDA or JAWS (recommended but not required)
