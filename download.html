What's Included

The zip contains everything you need to run the Chatyour client:

  • client.py — The main Chatyour client application
  • requirements.txt — List of Python dependencies to install

Installation Instructions

  1. Extract the zip file to a folder of your choice.
  2. Open a terminal in that folder and run: pip install -r requirements.txt
  3. Run the client: python client.py
  4. Optionally, use --ws for WebSocket mode if connecting to a remote server.

System Requirements

  • Windows operating system (7 or later)
  • Python 3.8 or higher
  • wxPython 4.x
  • websockets library (for WebSocket mode)
  • accessible_output2 for text-to-speech feedback
  • A screen reader such as NVDA or JAWS (recommended but not required)
