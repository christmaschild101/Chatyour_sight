About the Message of the Day

The Message of the Day (MOTD) is a feature of the Chatyour server that allows administrators to broadcast important information to all connected users. When set, the MOTD is displayed automatically upon login and can include:

  • Server announcements — Maintenance windows, new features, or policy changes.
  • Community news — Events, updates, or highlights from the chat community.
  • Important alerts — Security notices, downtime warnings, or version updates.

Now that the client application is available and the server is active, the administrator will use the MOTD feature to keep everyone informed. For now, no message has been posted.

For Administrators

If you are a Chatyour server administrator, you can set the Message of the Day through the admin console in the client application. The MOTD is stored server-side and persists across restarts.
